# Поисковый агент с веб-интерфейсом

Упрощенный поисковый агент на LangChain с веб-интерфейсом.

## Функционал
- Поиск актуальной информации в интернете через Serper API
- Веб-интерфейс с реальным временем общения

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите агента: 
```bash
python main.py
```
4. Откройте страницу в браузере
http://localhost:8000/
